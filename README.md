# 这个将是我们的可证明公平随机抽奖合约
## About
这些代码是用来创建可证明公平的智能合约彩票的

## what we want it to do
1. 用户可以通过购买一张彩票来参加抽奖
2. 彩票的费用将在抽奖时全部给赢家 
3. 经过X时间周期 彩票将自动抽出一个赢家 这将通过编程方式完成
4. 我们将使用chailink VRF和chainlink Automation来完成这个任务
    1. chainlink VRF -> Randomness
    2. chainlink Automation -> time based trigger
